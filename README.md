## Завдання 1 — Рекурсивне копіювання та сортування файлів

Створити програму на Python, яка:

- рекурсивно обходить вихідну директорію;
- копіює всі файли в директорію призначення;
- сортує файли у піддиректорії за типами (за розширенням).

### ✨ Критерії виконання

#### **1. Парсинг аргументів**
Скрипт приймає два аргументи командного рядка:
- шлях до вихідної директорії;
- шлях до директорії призначення (за замовчуванням — `dist`, якщо аргумент не передано).

#### **2. Рекурсивне читання директорій**
Функція повинна:

- приймати шлях до директорії як аргумент;
- перебирати всі елементи директорії;
- якщо елемент — *директорія*, викликати себе рекурсивно;
- якщо елемент — *файл*, передавати його на обробку для копіювання.

#### **3. Копіювання файлів**
Для кожного файлу:

- створюється піддиректорія на основі розширення файлу;
- файл копіюється у відповідну піддиректорію.

#### **4. Обробка винятків**
Програма повинна коректно обробляти:

- помилки доступу;
- помилки читання файлів;
- неіснуючі шляхи.

#### **5. Результат**
Після завершення роботи:

- усі файли вихідної директорії рекурсивно скопійовані у нову директорію;
- файли розсортовані за розширеннями у відповідних підпапках.

## Завдання 3 — Ханойські вежі

Написати програму, яка моделює переміщення дисків у класичній задачі **Ханойські вежі**:

- Диски знаходяться на стрижні **A**.
- Мета — перемістити їх на стрижень **C**.
- Стрижень **B** використовується як допоміжний.

Диски мають різний розмір і спочатку розташовані у порядку зменшення (зверху вниз).

### ✨ Правила

1. За один хід можна перемістити лише один диск.
2. Більший диск не можна класти на менший.

### Вхідні дані

`n` — кількість дисків.

### Вихідні дані

Логування кожного кроку переміщення, включаючи поточний стан трьох стрижнів.
